Update April 14, 2022
Due to “great service comes with great bills” I closed down account on AWS, the main reason I was fiddling around on AWS just because when I apply for a job some companies requires you to have some AWS experience, or at least some familiarities. Once I used it to deploy my project from my local development, then using Code Deploy, Code Pipeline, EC2, RDS, EB, S3, setting up IAM to remote the DB using Workbench…I do not find anything extra ordinary in terms of using it, unless you’re truly DevOps maybe. Don’t get me wrong the services are great, fast, etc…but putting it as a must skills to have on job desc is a bit overrated. Please feel free to leave comment if you have any thoughts about AWS would love to know your opinions.”
